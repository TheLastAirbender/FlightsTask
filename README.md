# FlightsTask
Тестовое задание

### Сборка и запуск приложения
**Необходимо:**
1. Java 17
2. Gradle
3. Git

```shell script
# Склонировать проект к себе
git clone https://github.com/devalurum/egrp365-bot.git

# загружает gradle wrapper
gradlew wrapper

# сборка проекта
gradlew clean build 

# запуск jar-файла
java -jar build/libs/FlightsTask.jar
```
