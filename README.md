# FlightsTask
Тестовое задание

### Сборка и запуск приложения
**Необходимо:**
1. Java 11
2. Gradle
3. Git

```shell script
# Склонировать проект к себе
git clone https://github.com/TheLastAirbender/FlightsTask.git

#перейти в каталог
cd FlightsTask

# загружает gradle wrapper
gradle wrapper

# сборка проекта
gradle clean build 

# запуск jar-файла
java -jar build/libs/FlightsTask.jar
```
